WithLog+
ひとりじゃない、毎日のログ。


## サービス概要
WithLog+ は、毎日の TODO・習慣・日記・体重・育児記録をひとつにまとめて管理できる、
バディ付きライフログアプリです。
忙しい日々の中でも「今日のこと」が一画面で把握でき、「今日もやりきった」と思える体験をサポートします。

---

## このサービスへの思い・作りたい理由
育児や家事、仕事に追われる毎日の中で、
「TODOや日記をつけようとしても、結局続かない」
「やることはたくさんこなしているのに、誰にも褒められない」
と感じることが多くありました。

TODO管理アプリ、日記アプリ、体重管理アプリなどは数多く存在しますが、
それぞれが別々で、設定やカスタマイズが難しく、
結果的にどれも続かなかったという経験があります。

忙しいからこそ、
「ここを開けば今日のことが全部わかる」
そんなアプリがあればいいのに、と感じたことが
本サービスを企画したきっかけです。

天気や洗濯の目安、今日のTODO、ちょっとしたメモや日記、
習慣記録、体重管理を一箇所にまとめて管理できることを大切にしました。
また、忙しい日々の中で忘れがちな子どもの成長記録も、
思いついた時に気軽に残せるようにしています。

ただ記録するだけでは続かないことも多く、
毎日頑張っていても誰にも気づいてもらえないのは少し寂しい。

そこで、日々の行動や変化に気づき、
「今日もよく頑張ったね」と声をかけてくれる
バディ機能を取り入れました。

WithLog+ は、忙しい毎日の中で
少しだけ自分を肯定できる時間をつくれる存在を目指しています。

---

## ユーザー層について
主なターゲットは、日々複数のタスクをこなしながら忙しく生活している人です。

- 家事・育児・仕事など、日常のタスクに追われている人
- TODO管理や日記、健康管理などを複数のアプリで分散して管理している人
- ウィジェットや細かいカスタマイズが苦手で、シンプルに使えるアプリを求めている人
- 記録や習慣を続けたい気持ちはあるが、途中で挫折しやすい人
- 自分の頑張りを肯定してくれる存在が欲しいと感じている人

「忙しい毎日を一つの場所で管理したい」という課題は
性別やライフステージを問わず共通するものだと考えています。

---

## サービスの利用イメージ
ユーザーは、朝やスキマ時間にアプリを開くことで、

- 今日の日付・天気・洗濯指数を確認
- 今日のTODOや習慣をチェック
- メモや日記を簡単に記録
- 体重の変化を確認
- バディからの一言メッセージで前向きな気持ちになる

といった一連の流れを、ホーム画面だけで完結できます。

また、ママ・パパ向け機能として
子どもの成長記録を月齢単位で残すことも可能です（ON / OFF切り替え可）。

バディが日々の頑張りや変化に反応することで、
「今日もやり切った」と感じられる体験を提供します。

---

## サービスの差別化ポイント・推しポイント
WithLog+ の最大の特徴は、
複数の役割を持つ機能を「一画面・一アプリ」に集約している点です。

TODO管理、日記、健康管理、育児記録は
それぞれ専用アプリが多く存在しますが、
忙しい人ほどアプリを切り替えること自体が負担になります。

WithLog+ では、
「今日を過ごすために必要な情報と記録」を一箇所にまとめ、
さらにバディ機能によって
行動や継続を肯定的にサポートする点を差別化ポイントとしています。

また、習慣や体重管理に対して
ご褒美付き設定を任意で有効にできる仕組みを取り入れ、
楽しみながら続けられる設計にしています。

### 既存サービスとの違い
- TODO・習慣・日記・体重管理を1画面に集約
- 設定やカスタマイズを最小限に抑えたシンプル設計
- バディによる一言メッセージで心理的ハードルを下げる
- 子どもの成長記録を「非公開前提」で残せる

多機能ではなく、
**「毎日開きたくなること」** を最優先に設計しています。

## 機能候補

### MVPリリース時に実装したい機能
- ユーザー登録 / ログイン機能
- ホーム画面
  - 日付表示
  - 天気・洗濯指数の表示
  - バディキャラクターとメッセージ表示
- TODO管理機能
  - 今日のTODOの追加・編集・完了
- メモ / 日記機能
  - 日付ごとの記録
  - 一覧表示・詳細表示
- 習慣管理機能
  - 習慣の登録・編集
  - 期間設定（開始日・終了日）
- 体重管理機能
  - 日付ごとの体重入力
  - グラフ表示
- 子どもの成長記録機能（ON / OFF可能）
  - 月齢ごとの記録
  - 子ども切り替え機能
- 設定機能
  - 目標体重の設定
  - 子ども情報の設定
  - 成長記録機能のON / OFF

### 本リリースまでに追加したい機能
- TODOの時間帯分類（朝 / 日中 / 夜）
- TODOカテゴリ機能（家事・育児・仕事・健康など）
- 習慣・体重管理へのご褒美付き設定（ON / OFF）
- 習慣達成時のポップアップ表示
- 習慣・ご褒美達成のSNSシェア機能
- バディメッセージのバリエーション追加
- UI / UXの改善

### CRUD以外の実装イメージ
- 天気情報は外部APIから取得し、当日分を表示
- 体重管理は日付ごとのデータを保存し、グラフとして可視化
- バディ機能は、ユーザーの行動結果や状態に応じた
  あらかじめ用意したメッセージを表示する形で実装予定
