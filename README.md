# WithLog+
**ひとりじゃない、毎日のログ。**

---

## サービス概要
WithLog+ は、毎日の TODO・習慣・日記・体重・育児記録を、
<br>ひとつの画面・ひとつのアプリでまとめて管理できるライフログアプリです。<br>
日々の必要なタスクを一箇所で把握でき、バディ機能によって継続や満足感をサポートします。

---

## このサービスへの思い・作りたい理由
育児や家事、仕事に追われる毎日の中で、

- やりたいこと、やらなければいけないことは多いのに、タスクを管理しきれない  
- TODOや日記をつけようとしても、結局続かない  
- たくさん頑張っているのに、誰にも気づいてもらえない  

と感じることが多くありました。

TODO管理、日記、体重管理などのアプリは数多く存在しますが、  
それぞれが別々で設定やカスタマイズも複雑なものが多く、  
結果として **どれも続かなかった** という経験があります。

忙しいからこそ、

**「ここを開けば、今日のことがすべてわかる」**

そんなシンプルなアプリがあればいいのに、  
と感じたことが、本サービスを企画したきっかけです。

WithLog+では、天気や洗濯の目安、今日のTODO、ちょっとしたメモや日記、  
習慣記録や体重管理といった“1日を過ごすために必要な情報”を<br>
**一箇所にまとめて管理** できることを大切にしました。

また、忙しい日々の中で忘れがちな  
**子どもの成長記録** も、思いついた時に気軽に残せる設計にしています。

ただ記録するだけでは続かないことも多く、  
毎日頑張っていても誰にも気づいてもらえないのは少し寂しい。

そこで、日々の行動や変化に気づき、  
「今日もよく頑張ったね」と声をかけてくれる  
**バディ機能** を取り入れました。

WithLog+ は、  
忙しい毎日の中で **少しだけ自分を肯定できる時間** をつくれる存在を目指しています。

## ユーザー層について

### メインターゲット
家事・育児・仕事など日常のタスクを抱え、忙しさからキャパオーバーになりやすい
子育て中の20〜40代。

### サブターゲット
仕事や私生活のタスクが多く、やりたい習慣や記録も含めて日々のことを整理しきれずにいる社会人。

---

## サービスの利用イメージ
ユーザーは、朝やスキマ時間にアプリを開くことで、

- 今日の日付・天気・洗濯指数を確認  
- 今日のTODOや習慣をチェック  
- メモや日記を簡単に記録  
- 目標体重までの残りの日数と体重を確認
- バディからの一言メッセージで前向きな気持ちになる  

といった一連の行動を、ホーム画面を起点として行うことができます。

また、ママ・パパ向け機能として  
子どもの成長記録を **月齢単位で記録** することも可能です  
（機能のON / OFF切り替え対応）。

バディが日々の頑張りや変化に反応することで、  
「今日もやりきった」と感じられる体験を提供します。

---

## サービスの差別化ポイント・推しポイント
WithLog+ の最大の特徴は、  
複数の役割を持つ機能を **「一画面・一アプリ」** に集約している点です。

TODO管理、日記、健康管理、育児記録は  
それぞれ専用アプリが多く存在しますが、  
忙しい人ほど **アプリを切り替えること自体が負担** になります。

WithLog+ では、

- 「今日を過ごすために必要な情報と記録」を一箇所に集約  
- バディ機能によって行動や継続を肯定的にサポート  

することで、記録そのものの心理的ハードルを下げることを
差別化ポイントとしています。

また、習慣や体重管理に対して  
**ご褒美付き設定（任意）** を用意し、  
楽しみながら続けられる設計を取り入れています。

### 既存サービスとの違い
- TODO・習慣・日記・体重管理を1画面に集約  
- 設定やカスタマイズを最小限に抑えたシンプル設計  
- バディによる一言メッセージで心理的ハードルを下げる  
- 子どもの成長記録を「非公開前提」で残せる  

多機能ではなく、  
**「毎日開きたくなること」** を最優先に設計しています。

## 機能候補

### MVPリリース時に実装したい機能
- ユーザー登録 / ログイン機能  
- ホーム画面  
  - 日付表示  
  - 天気・洗濯指数表示  
  - バディキャラクターとメッセージ表示  
- TODO管理機能  
  - 今日のTODOの追加・編集・完了  
- メモ / 日記機能  
  - 日付ごとの記録  
  - 一覧表示・詳細表示  
- 習慣管理機能  
  - 習慣の登録・編集  
  - 期間設定（開始日・終了日）  
- 体重管理機能  
  - 日付ごとの体重入力  
  - グラフ表示  
- 子どもの成長記録機能（ON / OFF可能）  
  - 月齢ごとの記録  
  - 子ども切り替え機能  
- 設定機能  
  - 目標体重設定  
  - 子ども情報設定  
  - 成長記録機能ON / OFF  

### 本リリースまでに追加したい機能
- TODOの時間帯分類（朝 / 日中 / 夜）  
- TODOカテゴリ機能（家事・育児・仕事・健康など）  
- 習慣・体重管理へのご褒美付き設定  
- 習慣達成時のポップアップ表示  
- 習慣・ご褒美達成のSNSシェア機能  
- バディメッセージのバリエーション追加  
- UI / UXの改善

## 使用する技術スタック

### 開発環境
- Docker

### フロントエンド
- ERB  
- JavaScript（必要に応じて）

### 認証
- Devise  

### インフラ / デプロイ
- Render  

### その他使用予定
- GitHub（バージョン管理）  
- GitHub Actions（CI / テスト自動化）  
- Chart.js（体重グラフ表示）  
- 外部API（天気情報取得）  

### CRUD以外の実装イメージ
- 天気情報は外部APIから取得し、当日分を表示  
- 体重管理は日付ごとのデータを保存し、グラフとして可視化  
- バディ機能はユーザーの行動結果や状態に応じて  
  あらかじめ用意したメッセージを表示する形で実装予定
